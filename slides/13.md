# How to Actually Start

Install Claude Code — it's one terminal command.

Docs: ==docs.anthropic.com/en/docs/claude-code/overview==

Your first prompt:

```
> Look at the files in this folder. Find contradictions,
  inconsistencies, or anything that doesn't add up.
```

You don't need to know how to code.
You type English — it does the rest.
