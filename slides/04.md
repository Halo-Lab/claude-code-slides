# The Terminal: Not Scary, I Promise

A terminal is just a folder browser without the pictures. Same files, same folders — you just type instead of click.

Four commands. That's all you need to start.

```
$ ls              # list files in a folder
$ cd folder-name  # go into a folder
$ mkdir name      # create a new folder
$ claude          # start Claude Code
```

A good terminal to start with is Ghostty — ==ghostty.org==. It's free, looks clean, zero configuration effort. But any terminal works.
