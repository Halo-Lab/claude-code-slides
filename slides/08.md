# Slash Commands: Teach Claude ==Your== Workflow

Claude Code supports custom commands — reusable prompts you write once and run anytime with ==/command-name==.

```
.claude/commands/research.md     # /research
.claude/commands/plan.md         # /plan
.claude/commands/spec.md         # /spec
```

Here's a snippet from my <a href="https://github.com/Halo-Lab/claude-code-slides/blob/main/.claude/commands/research.md" style="color:var(--yellow)">/research</a> command:

```
# /research - Gather External Information

## Process
1. Clarify the question
2. Search (WebSearch, WebFetch)
3. Evaluate sources by credibility
4. Synthesize findings into a structured report
5. Save to docs/research/ for future reference
```
